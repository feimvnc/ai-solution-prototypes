# 🎹 AI Typing Tutor - Master Your Keyboard with AI

A modern, professional typing tutor web application powered by local Ollama AI to help you improve your typing speed and accuracy without looking at the keyboard.

## ✨ Features

### Core Features
- 📚 **Classical Literature Practice** - Type passages from famous books
- 🎯 **Key Combination Drills** - Focus on specific key groups (home row, top row, etc.)
- 🤖 **AI-Generated Practice** - Custom texts generated by Ollama AI
- ⌨️ **Virtual Keyboard** - Real-time visual feedback on key presses
- 📊 **Detailed Statistics** - WPM, accuracy, errors, and time tracking
- 🎨 **Modern UI** - Fluid animations and professional design

### AI-Powered Features
- 💡 **Personalized Tips** - AI analyzes your typing patterns and provides custom advice
- 📈 **Performance Analysis** - AI evaluates your session and suggests improvements
- ✍️ **Custom Text Generation** - AI creates practice texts based on difficulty level
- 🎓 **Smart Coaching** - Identifies problem keys and provides targeted exercises

## 🚀 Quick Start

### Prerequisites

1. **Python 3.8+**
2. **Ollama** installed and running
3. **Qwen2.5 model** (or any compatible model)

### Install Ollama

```bash
# macOS/Linux
curl -fsSL https://ollama.com/install.sh | sh

# Or download from: https://ollama.com/download
```

### Pull AI Model

```bash
ollama pull qwen2.5:3b
```

### Installation

1. **Setup Backend**

```bash
cd backend
python3 -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
pip install -r requirements.txt
```

2. **Start Backend Server**

```bash
python app.py
```

The API will run on `http://localhost:5000`

3. **Start Frontend**

Use Finder to find the file, drag to a browser, you should see the page.
Open `frontend/index.html` in your browser, or use a simple HTTP server:

```bash
cd frontend
python3 -m http.server 8000
```

Then visit `http://localhost:8000`

## 🔧 Configuration

### Backend API Endpoints

- `GET /api/health` - Check API and Ollama status
- `GET /api/texts` - Get all practice texts
- `GET /api/texts/<id>` - Get specific text
- `POST /api/ai/tips` - Get personalized typing tips
- `POST /api/ai/generate-text` - Generate custom practice text
- `POST /api/ai/analyze` - Analyze typing session
- `POST /api/stats` - Save statistics

### Customization

**Add Your Own Texts:**
Edit `data/texts/classical_texts.json`:

```json
{
  "id": 9,
  "title": "Your Text Title",
  "difficulty": "medium",
  "text": "Your practice text here..."
}
```

**Change AI Model:**
Edit `backend/app/services/ollama_service.py`:

```python
def __init__(self, model="your-model-name"):
```

**Adjust Colors:**
Edit `frontend/styles/main.css` CSS variables:

```css
:root {
    --primary-color: #6366f1;
    --secondary-color: #8b5cf6;
    /* ... */
}
```

## 🐛 Troubleshooting

### Ollama Not Connected
- Ensure Ollama is running: `ollama serve`
- Check model is installed: `ollama list`
- Verify connection: `curl http://localhost:11434/api/tags`

### CORS Errors
- Make sure backend is running on port 5000
- Check browser console for specific errors
- Verify `flask-cors` is installed

### AI Features Not Working
- Confirm Ollama is running
- Check backend logs for errors
- Ensure qwen2.5:3b model is pulled

### Frontend Not Loading
- Check browser console for JavaScript errors
- Verify all JS files are loaded
- Try hard refresh (Ctrl+Shift+R)

## 🎯 Tips for Best Results

1. **Posture** - Sit up straight, feet flat on floor
2. **Hand Position** - Keep fingers on home row (ASDF JKL;)
3. **Don't Look** - Trust the virtual keyboard, not your physical one
4. **Accuracy First** - Speed comes naturally with practice
5. **Regular Practice** - 15-20 minutes daily is better than long sessions
6. **Use AI Tips** - Follow the personalized advice from the AI coach

## 📝 Development

### Tech Stack
- **Backend:** Flask, Python 3
- **AI:** Ollama with Qwen2.5
- **Frontend:** Vanilla JavaScript (no frameworks)
- **Styling:** Custom CSS with CSS Grid and Flexbox
- **Fonts:** Inter (Google Fonts)

### Adding New Features

1. **New Practice Mode:**
   - Add mode button in HTML
   - Create mode handler in `app.js`
   - Add corresponding UI section

2. **New AI Feature:**
   - Add method in `ollama_service.py`
   - Create API endpoint in `routes.py`
   - Call from frontend in `app.js`

3. **New Statistics:**
   - Update `TypingEngine` class in `typing.js`
   - Add display elements in HTML
   - Update stats rendering in `app.js`
